###1、已知一棵二叉树按顺序存储结构进行存储、设计一个算法，求编号分别为i和j的两个节点的最近公共祖先节点的值
```c
typedef struct SqTree{
    ElemType data[MaxLength];
}SqTree;
ElemType Comm_Ancestor(SqTree T,int i,int j){
    if(T.data[i]!='#'&&T.data[j]!='#'){
        while(i!=j){
            if(i>j) i=i/2;
            else j=j/2;
        }
        return T.data[i];
    }
}
```
